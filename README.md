## 📋 عن المشروع

**جدولي** منصة ويب متخصصة تساعد أصحاب الحضانات والمعلمات على إدارة عملهم باحترافية من خلال:

- **ملفات PDF** جاهزة للتحميل الفوري (جداول، سجلات، نماذج تقييم)
- **كورسات تدريبية** في إدارة الحضانات، استراتيجيات التعلم، وتصميم الوسائل التعليمية
- **خطط تعليمية** وترفيهية متكاملة وجاهزة للاستخدام

## 🗂️ محتوى المنصة

### الملفات الرقمية
- الجدول الأسبوعي العملي لتنظيم اليوم الافتراضي
- حزمة الملفات الإدارية المتقدمة للحضانات
- الحقيبة المتكاملة للأنشطة الترفيهية الصيفية
- دليل ركن التعايش والقراءة التفاعلية

### الكورسات التدريبية
- دورة استراتيجيات التعلم للأطفال
- دورة التميز الإداري وتأسيس الحضانات
- ورشة تصميم الوسائل التعليمية المبتكرة
- دورة التخطيط الإستراتيجي للأنشطة الصيفية

## 🚀 تشغيل المشروع محلياً

```bash
# استنساخ المشروع
git clone https://github.com/mo7amedxv/jadwaly.git

# فتح المجلد
cd jadwaly

# فتح الملف مباشرة في المتصفح
open index.html
```

لا يحتاج المشروع لأي dependencies أو build steps.
